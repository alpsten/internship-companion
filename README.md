# Internship Checklist

Helping Swedish LIA students plan, track, and land their internships with clarity, momentum, and a calm mind.

This repository contains:
- A content-driven checklist and resource hub (guides, templates, trackers)
- A React/TypeScript frontend (coming soon) that surfaces the checklist in an interactive way
- Documentation for collaborators (`docs/`)

## Project Status
- **Phase:** Discovery & planning
- **Focus:** Validate information architecture, agree on MVP scope, ship a visible frontend shell quickly
- **Primary language:** Swedish (English toggle planned)

## Quick Links
- `docs/product-brief.md` – goals, users, success measures, MVP scope
- `docs/content-outline.md` – structured content plan derived from early notes
- `docs/CODEX.md` – engineering principles and tech stack policy
- `docs/AGENTS.md` – collaboration agreements and decision flow

## Getting Started
We will scaffold the frontend with React + TypeScript and pnpm. Once committed, a development shell will be available via:
1. `pnpm install`
2. `pnpm dev`

Hosting target: GitHub Pages (static export) with the option to expand later.

## Contributing
- Read `docs/CODEX.md` and `docs/AGENTS.md` before touching code.
- Discuss any change in direction with @alpsten.
- File issues for new content sections, translations, or automation ideas before implementing.

## Roadmap at a Glance
1. Finalize product brief and content structure.
2. Create wireframes and interaction notes for the checklist experience.
3. Scaffold the frontend, deploy a skeleton page to GitHub Pages for rapid feedback.
4. Iteratively add content modules (guides, templates, trackers) and localization support.
