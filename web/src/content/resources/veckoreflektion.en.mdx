## 15-minute loop every week
1. Filter your application log to the past week.
2. Note three things that moved forward (replies, new contacts, better messaging).
3. Write one thing you want to tweak next week.

## Questions to answer
- What triggered the positive responses?
- Which step did I postpone—and why?
- Which companies feel most promising right now?

## Micro plan
```
Monday: update log + plan outreach
Wednesday: follow up + call one company
Friday: weekly reflection + celebrate a micro win
```

## Bonus
- Block a recurring 15-minute calendar slot so you never skip the reflection.
- Send a short update to a friend or mentor—accountability keeps momentum.
